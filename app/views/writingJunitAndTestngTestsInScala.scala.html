Writing JUnit and TestNG tests in Scala

One way to integrate ScalaTest with existing tools is to run your ScalaTest tests through JUnit. With ScalaTest's JUnitRunner annotation, you can run any ScalaTest Suite written in any style with JUnit 4.

If you would like to write native JUnit or TestNG tests in Scala, see:

  • Getting started with JUnit 4 in Scala
  • Getting started with TestNG in Scala
  • Getting started with JUnit 3 in Scala
